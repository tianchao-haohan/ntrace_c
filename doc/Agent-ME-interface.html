Http request url: http://host:port/post-command

1 Add agent

1.1 request:

{
    "command":"add-agent"
    "body":{
        "ip":"192.168.1.100"
        "port":"4499"
        "agent-id":"1"
    }
}

1.2 response:

{
    "code":"0"
    "body":{
        "status":"ready"
    }
}

2 Remove agent

        "agent-id":"1"
    }
}

{
    "code":"0"
}
